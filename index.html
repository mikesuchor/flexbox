Display

Below is an inline flexbox container, it is an inline element and takes up the width of its flex items.
On hover, its a block flexbox container and takes up the width of the whole page.

display: inline-flex;display: flex;

Flex-Direction

Below is the default flex-direction property, with main axis left to right and cross axis top to bottom.
On hover, its direction is reversed, with main axis right to left and cross axis bottom to top.

flex-direction: row;flex-direction: row-reverse;

Flex-Wrap

Below is the default flex-wrap property, with flex items fitting themselves into 1 row.

Flex & Order

The flex property specifies the length of the flex item relative to the other flex items in the same container.
The order property specifies the order of the flex item relative to the other flex items in the same container.

flex: <#>;order: <#>;

Justify-Content

The justify-content property aligns the flex items on the main axis, aligned by default to the left.
On hover, the flex items are aligned to the right.

justify-content: flex-startjustify-content: centerjustify-content: flex-end
